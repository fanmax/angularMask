# angularMask

Input mask

## Requirements

- AngularJS

## Usage

You can get it npm
```sh
npm install angular-sky-mask --save
```

####using 
var app = angular.module('app', ['skyang.mask',...]); 

###Example:
MASK
input type="text" class="form-control" skyangMask="999.999.999-99"

MONEY
input type="text" class="form-control" skyangMask="money"
